¿Qué es Aiko Hero y por qué debería usarlo?

Aiko Hero es una librería basada en TypeScript que te ayuda a crear o añadir heros a tu página web de una manera muy sencilla.

Diseño

Nuestro diseño está preparado para combinar perfectamente con cualquier plataforma.

Configurable

Tendrás muchas opciones para configurar tu hero de acuerdo a tus propios requerimientos.

Responsive

Podrás usar AikoHero desde cualquier dispositivo móvil o de escritorio.
